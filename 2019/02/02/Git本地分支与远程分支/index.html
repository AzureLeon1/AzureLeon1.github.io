Git本地分支与远程分支

Posted on 2019-02-02 | In Git

把远程仓库中已经有的dev分支同步到本地：

git checkout -b dev 新建并切换到本地dev分支

git pull origin dev 本地分支与远程分支相关联

在本地新建dev分支并推送到远程：

git checkout -b dev

git push origin dev:dev 这样远程仓库中也就创建了一个dev分支。冒号左边是本地分支，右边是远程分支。
