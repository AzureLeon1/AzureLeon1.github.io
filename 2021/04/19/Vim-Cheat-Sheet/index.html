    移动光标

    • h, j, k, l - 光标左移、下移、上移、右移
    • w - 移动到下个单词开头
    • e - 移动到下个单词结尾
    • b - 移动到上个单词开头
    • 0 - 移动到行首
    • ^ - 移动到行首的非空白符
    • $ - 移动到行尾
    • g_ - 移动到行内最后一个非空白符
    • H - 移动到当前页面顶部
    • M - 移动到当前页面中间
    • L - 移动到当前页面底部
    • gg：移动到文件第一行
    • G - 移动到文件最后一行
    • 5gg or 5G - 移动到第五行
    • } - 移动到下一个段落 (当编辑代码时则为函数／代码块)
    • { - 移动到上一个段落 (当编辑代码时则为函数／代码块)
    • Ctrl + d - 向下滚动半屏
    • Ctrl + u - 向上滚动半屏
    • zz - 移动屏幕使光标居中
    • % - 跳转到配对的符号(默认支持的配对符号组:: ‘()’, ‘{}’, ‘[]’ - 在vim中使用 :h matchpairs 获得更多信息)

    Tip 命令前追加数字表示命令的重复次数, 比如 4j 表示向下移动四行、d2w表示向后删除两个单词

    插入模式 - 插入/追加文本

    • i - 在光标前插入
    • a - 在光标后插入
    • I - 在行首插入
    • A - 在行尾插入
    • o - 在当前行之下另起一行, 开始插入字符
    • O - 在当前行之上另起一行, 开始插入字符

    编辑文本

    • u - 撤销最后执行的命令
    • U - 撤销对整行的修改
    • ctrl + R - 重做被撤销的命令
    • r - 替换当前字符
    • R - 连续替换多个字符
    • c [number] motion - 更改（先删除到motion位置，然后进入insert模式）
      • cw/ce - 从光标位置开始, 修改单词
      • c$ - 从光标位置开始，修改本行
    • . - 再次执行上个命令

    选择文本（可视化模式）

    • v - 进入可视化模式, 移动光标高亮选择, 然后可以对选择的文本执行命令(比如y-复制)
    • V - 进入可视化模式(行粒度选择)
    • Ctrl + v - 进入可视化模式(矩阵选择)

    寄存器

    • :reg[isters] - 显示寄存器内容
    • "xy - 复制内容到寄存器 x
    • "xp - 粘贴寄存器 x 中的内容
    • "+y - 复制内容到系统剪贴板寄存器
    • "+p - 粘贴系统剪贴板寄存器的内容

    Tip 寄存器被存储在 ~/.viminfo 中, 在下次重启vim时仍会加载

    剪切, 复制, 粘贴

    • x - 剪切当前字符
    • d [number] <motion> - 剪切
      • de - 从光标位置剪切到当前词的词尾
      • dw - 从光标位置剪切到下一词的词首
      • d$/D - 从光标位置剪切到行尾
      • dd - 剪切当前行
      • 2dd/d2d - 剪切两行
    • y - 复制
      • yy - 复制当前行
      • 2yy - 复制 2 行
      • yw - 复制当前单词
      • y$ - 复制, 从光标位置到行末
    • p - 在光标后粘贴
    • P - 在光标前粘贴

    查找/替换

    • /pattern - 查找pattern

    • ?pattern - 向上查找pattern

    • n - 下一个

      如果查找已经到达文件末尾，查找会自动从文件头部继续查找，除非 ‘wrapscan’ 选项被复位。

    • N - 上一个

    • :s/old/new - 替换光标所在行的第一个匹配串

    • :s/old/new/g - 替换光标所在行的全部匹配串

    • :#,#s/old/new/g - 替换两行之间的全部匹配串，#表示行号

    • :%s/old/new/g - 替换文件中的所有匹配串

    • :%s/old/new/gc - 逐个替换文件中的所有匹配串

    文字缩进

    • >> - 将当前行向右缩进，宽度由 shiftwidth 控制
    • << - 将当前行向左缩进，宽度由 shiftwidth 控制
    • == - 自动缩进

    退出

    • :w - 保存
    • :w [filename] - 保存到文件，并命名文件
    • :w !sudo tee % - 使用 sudo 保存当前文件
    • :wq or :x or ZZ - 保存并退出
    • :q - 退出(修改未保存时警告)
    • :q! or ZQ - 不保存强制退出
    • :wqa - 保存所有标签页并全部退出

    设置

    • :set [xxx] - 设置xxx选项
      • :set ic
      • :set hls
      • :set is
    • :set no[xxx] - 关闭选项
      • :set noic
      • :set nohlsearch

    多文件编辑

    • Ctrl + ws - 水平分割窗口
    • Ctrl + wv - 垂直分割窗口
    • Ctrl + ww - 在窗口间切换
    • Ctrl + wq - 关闭窗口
    • Ctrl + wx - exchange current window with next one
    • Ctrl + w= - make all windows equal height & width
    • Ctrl + wh - 切换到左侧窗口
    • Ctrl + wl - 切换到右侧窗口
    • Ctrl + wj - 切换到下侧窗口
    • Ctrl + wk - 切换到上侧窗口

    多行同时编辑

    1. ctrl + v 进入 visual block 模式

    2. 选中block

    3. I 进入插入模式

    其他

    • :![shell command] - 执行外部命令
    • :r [filename] - 插入另外的文件的内容
    • v [motion] :w [FILENAME] - 将当前编辑文件中可视模式下选中的内容保存到文件FILENAME中

    Nerd Tree插件

    • ,nn - 打开关闭目录树
    • q 关闭 NERDTree
    • o 在已有窗口中打开文件或目录，并将光标跳到该窗口
    • go 在已有窗口中打开文件或目录，但不跳到该窗口
    • O 递归打开选中 结点下的所有目录
    • t 在新 Tab 中打开选中文件/书签，并跳到新 Tab
    • T 在新 Tab 中打开选中文件/书签，但不跳到新 Tab
    • i split 一个新窗口打开选中文件，并跳到该窗口
    • gi split 一个新窗口打开选中文件，但不跳到该窗口
    • s vsplit 一个新窗口打开选中文件，并跳到该窗口
    • gs vsplit 一个新 窗口打开选中文件，但不跳到该窗口
    • x 合拢选中结点的父目录
    • X 递归 合拢选中结点下的所有目录
    • p 跳到父结点
    • P 跳到根结点
    • u 设置上级目录为根路径
    • U 设置上级目录为跟路径，但是维持原来目录打开的状态
    • r 刷新光标所在的目录
    • R 刷新当前根路径
    • I 显示或者不显示隐藏文件
    • f 打开和关闭文件过滤器
    • A 全屏显示 NERDTree，或者关闭全屏
    • C 将根路径设置为光标所在的目录
